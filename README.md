# Asset Ownership Transfer and Value Update in Hyperledger Fabric

## Problem

Implement a smart contract that allows:

1. Registering an asset
2. Transferring ownership
3. Updating asset value
4. Querying asset data

## Functions to Implement

RegisterAsset(assetID, owner, value)

TransferAsset(assetID, newOwner)

UpdateAssetValue(assetID, newValue)

GetAsset(assetID)
